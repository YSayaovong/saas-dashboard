# SaaS Dashboard (React)

This project is a SaaS‑style analytics dashboard built with React.  
It emphasizes the frontend: dashboard layouts, KPI visualization, reusable components, and responsiveness.

---

## Overview

The application simulates a SaaS product dashboard where users can view key metrics, navigate modules, and interact with UI elements. It demonstrates React-based UI development and responsive design.

---

## Tech Stack

- React
- JavaScript (ES6+)
- CSS / Responsive styling
- Client-side data handling

---

## Features

- Responsive dashboard layout for desktop, tablet, and mobile
- Metric cards, charts, and tabular UI components
- Sidebar and top navigation for multi-page structure
- Data fetching and rendering logic
- Modular React component architecture

## Getting Started

### 1. Clone repository
```bash
git clone https://github.com/YSayaovong/saas-dashboard.git
cd saas-dashboard
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm start
```

App will be accessible at:
```
http://localhost:3000
```

### 4. Build for production
```bash
npm run build
```
